The framework's features have different support levels across
runtimes.  This page is the **definitive reference**.

For each row:

- **✓** = supported, tested in CI.
- **⚠** = works, but not actively CI'd.
- **✗** = not supported.
- **N/A** = doesn't apply.


<Aside type="caution" title="Reading the warning marker">
  ```
  ⚠ = the framework hasn't seen production hours on this runtime
  for this feature.  Usually works; might have edge cases.
  ```
  If you encounter problems on a ⚠ feature, file a bug — most
  Deno-specific issues are simple fixes.
</Aside>

<Aside type="caution" title="Version pinning">
  ```json
  // package.json
  { "engines": { "bun": ">=1.1.30", "node": ">=22.12" } }
  ```
  Pin both Bun and Node engines if your code runs on either —
  prevents accidental dev/prod runtime drift.
</Aside>
